author: qoxop update at: 2018-12-19

JavaScript 有七种内置类型

空值（null）、未定义（undefined）、布尔值（ boolean）、数字（number）、字符串（string）、对象（object）、符号（symbol，ES6 中新增）

类型检测

    typeof undefined === "undefined"; // true
    typeof true === "boolean"; // true
    typeof 42 === "number"; // true
    typeof "42" === "string"; // true
    typeof { life: 42 } === "object"; // true
    typeof Symbol() === "symbol"; // true
    // 特殊
    typeof null === "object"; // true
    typeof function a(){ /* .. */ } === "function"; // true

特别的：

  • 函数的length属性是其声明的参数的个数

  • js中声明未赋值的变量未undefined，未声明就使用的标识符是Undeclared(会报错)， 但是 typeof Undeclared === 'undefined' 是 true

值

数组

js的数据元素可以是任何值，元素类型也可以不一致，甚至可以不设置(形成稀疏数组，即含有空白或空缺单元的数组)

    // 稀疏数组
    var _arr = []
    _arr[2] = 2;    // 等价于 [ , , 2]
    console.log(_arr, _arr[0], _arr[1]) // [empty x 2, 2], undefined, undefined
    var arr = [undefined, undefined, 2]
    console.log(_arr, _arr[0], _arr[1]) // [ undefined, undefined, 2], undefined, undefined
    _arr.forEach((item) => console.log(item)) // log 2 
    arr.forEach((item) => console.log(item))  // log undefined, undefined, 2

数字

实例方法
    var num = 10000
    // 指数形式字符串
    num.toExponential();
    // 可指定小数部分的显示位数， return string
    42..toFixed(3) // "42.000"
    42.toFixed(3) //  SyntaxError
    // 指定有效数位的显示位数
    num = 42.59;
    num.toPrecision( 1 ); // "4e+1"
    num.toPrecision( 2 ); // "43"
    num.toPrecision( 3 ); // "42.6"
    num.toPrecision( 6 ); // "42.5900"
类方法
    // 是否整数
    Number.isInteger( 42.000 ); // true
    Number.isInteger( 42.3 ); // false
    // 安全整数判断
    Number.isSafeInteger( Number.MAX_SAFE_INTEGER ); // true
    Number.isSafeInteger( Math.pow( 2, 53 ) ); // false
    // 是否为NaN
    Number.isNaN(..)
表达形式
  • 二进制 0b111
  • 八进制 0o111
  • 十六进制 0x111
  • 指数 1.798e+308
浮点数精度问题

0.1 + 0.2 === 0.3; // false

特别的数字
  • 最大浮点数：Number.MAX_VALUE, 大约是 1.798e+308
  • 最小浮点数：Number.MIN_VALUE, 接近于 0
  • 最大整数：Number.MAX_SAFE_INTEGER
  • 最小整数：Number.MIN_SAFE_INTEGER
  • 不是数字的数字：NaN, 将它理解为"无效数值" "失败数值" 或者 "坏数值", 一般是执行数学运算没有成功，这是失败后返回的结果，比如：2 / "second", 而且 NaN 非自反
  • 无穷数：Infinity, Number.POSITIVE_INfiNITY
  • 正负零值：一般于矢量计算时有用，判别，Object.is(-0, 0) // false

值与引用

值与引用其实很容易理解，因为js对于对象的值都是尽可能地只保存一份，不管是函数传参还是赋值，作用的都只是对象的引用。

封装对象

将基础类型包装成对象类型，从而可以执行对象操作

问题

包装对象的Boolean转化都是true，与它的原始类型不一样了

手动

通过原生构造函数， 常用的原生函数有：

  • String()
  • Number()
  • Boolean()
  • Array()
  • Object()
  • Function()
  • RegExp()
  • Date()
  • Error()
  • Symbol() —— ES6 中新加入的

自动

自动的包装有两种方式，一种的变量方式，另一种是常量方式。只有整数不支持常量方式的自动包装

    // 变量方式
    var a = 1;
    a.toString(); // "1", 这里自动包装了整数a
    // 常量方式
    "abc".charAt(0) // "2"

        console.log(b); // true
