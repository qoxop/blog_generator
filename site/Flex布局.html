Flex布局

Flex 是 Flexible Box 的缩写，意为"弹性布局"，用来为盒状模型提供最大的灵活性。 任何一个容器(包括行内元素)都可以指定为 Flex 布局。

注意，设为 Flex 布局以后，子元素的float、clear和vertical-align属性将失效。

1. 基本概念

采用Flex布局的元素称为Flex容器, 其子元素就自动变成了Flex项目

div.flexbox {
    display: flex;
}

Flex容器有两根轴—主轴和交叉轴，Flex项目就是依据这两个轴来进行布局的(包括调整大小与对齐方式)，默认的主轴是水平轴(→）,交叉轴是垂直轴(up to down)，轴起止与border。

2. 容器属性

flex-direction flex-wrap flex-flow justify-content align-items align-content

分别设置主轴方向，换行方式，主轴方向和换行方式，项目在主轴方式的分布规则，项目在交叉轴方向上的对齐方式，

2.1 flex-direction 属性

flex-direction属性决定主轴的方向（即项目的排列方向）。

.box {
  flex-direction: row | row-reverse | column | column-reverse; /* →  ←  ↓  ↑  */
}

2.2 flex-wrap属性

默认情况下Flex项目都排在了与主轴平行的轴线上，flex-wrap属性定义如果一条轴线排不下时如何换行。

  • nowrap(默认)：不换行，超出部分怎么显示有父元素决定
  • wrap：正常换行
  • wrap-reverse: 反向换行

2.3 flex-flow

flex-flow属性是flex-direction属性和flex-wrap属性的简写形式，默认值为row nowrap

2.4 justify-content属性

很多人喜欢将其定义为 " 项目在主轴方向上的对齐方式 ", 但个人喜欢理解为项目在主轴方式的分布规则，这个属性对于项目元素刚好排满一行主轴轴线时，并不会起任何作用，只有在排不满时，主轴轴线上出现了多余的空间， justify-content属性就是来描述如何分配这些空间的。

  • flex-start: 所有项目往主轴的起始位置靠，剩余的空间都在主轴的终止方向上
  • flex-end: 与flex-start相反
  • center: 所有项目往主轴的中间位置位置靠，剩余空间放两边
  • space-between: 两端元素贴壁(两端对齐)，剩余空间均匀分配给项目间的空隙
  • space-around: 每个项目两侧的间隔相等。所以，项目之间的间隔比项目与边框的间隔大一倍。

2.5 align-items属性

align-items属性定义同一行(或列)的项目在交叉轴的方向上如何对齐或拉伸。

  • flex-start：交叉轴的起点对齐。
  • flex-end：交叉轴的终点对齐。
  • center：交叉轴的中点对齐。
  • baseline: 项目的第一行文字的基线对齐。
  • stretch（默认值）：如果项目未设置高度或设为auto，将占满整个容器的高度或者与同行的最大高度保持一致。// 这里假设主轴是从左到右

2.6 align-content 属性

当 flex-wrap: wrap 且元素确实超出主轴宽度时，容器就有由超过一根的主轴轴线，align-content属性就是在描述各条主轴轴线在交叉轴方向上的分布规制，此属性有效的前提是 1. 在交叉反向上设置了长度 2. 其长度足够长(由剩余空间)

  • flex-start：与交叉轴的起点对齐。
  • flex-end：与交叉轴的终点对齐。
  • center：与交叉轴的中点对齐。
  • space-between：与交叉轴两端对齐，轴线之间的间隔平均分布。
  • space-around：每根轴线两侧的间隔都相等。所以，轴线之间的间隔比轴线与边框的间隔大一倍。
  • stretch（默认值）：轴线占满整个交叉轴。

3.项目属性

order flex-grow flex-shrink flex-basis flex align-self

  1. order：定义项目的排列顺序。数值越小，排列越前，默认为0
  2. flex-grow： 定义了项目如何分配主轴上的剩余空间，公式，a的宽度 = 剩余宽度 * a / (a+b+c+...)
  3. flex-grow：如果每个项目主轴方式上设置了固定的长度且flex-wrap: nowrap, 导致如果主轴长度不够时需要压缩项目，这个属性就是来分配压缩比例的。数值越大压缩越严重。
  4. flex-basis：定义项目在主轴上的初始宽度
  5. flex：flex-grow, flex-shrink 和 flex-basis的简写
  6. align-self： 修改单个项目在交叉轴方向上的对齐方式
